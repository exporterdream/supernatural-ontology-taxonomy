# 超自然实体分类树形图（Mermaid可视化）

本图展示"本源（Source）× 维度（Dimension）"双轴模型的完整层级结构。

```mermaid
graph TD
    A[超自然实体总界] --> B[自然本源 S-Nat]
    A --> C[灵魂本源 S-Soul]
    A --> D[生物本源 S-Bio]
    A --> E[异界本源 S-Cos]

    %% 自然本源分支
    B --> B1[D-2 实体态]
    B --> B2[D-3 化身态]
    B --> B2[D-3 化身态]
    B1 --> B1a[白骨精*]
    B1 --> B1b[牛魔王]
    B2 --> B2a[洪荒圣人]
    B2 --> B2b[玉皇大帝/高位存在]
    B2 --> B2c[尘世七执政·原神]

    %% 灵魂本源分支
    C --> C1[D-0 纯灵态]
    C --> C2[D-1 半虚态]
    C1 --> C1a[城隍/土地公]
    C1 --> C1b[香火神]
    C2 --> C2a[聂小倩]
    C2 --> C2b[贞子]

    %% 生物本源分支
    D --> D1[D-2 实体态]

    E1 --> E1b[混沌邪神·战锤40K]
    E2 --> E2a[外神投影]
```

> **说明**：  
> *白骨精的坐标标注为 (S-Nat, D-2)，但其物质基础源于人类尸骨，存在S-Soul的残留痕迹。详见[白骨精词条](./entities/bai-gu-jing.md)。  
> 本树形图将随新实体的收录而动态更新。同一实体如具有多维显形能力（如克苏鲁），可同时归属多个D代码。
