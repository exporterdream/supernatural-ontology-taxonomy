## 📌 关于“本体论”一词的说明

本体系所用“本体论（Ontology）”取其**信息科学引申义**——即“对特定领域的概念及概念间关系的明确规范”（Gruber, 1993），而非哲学形而上学中关于“存在本质”的研究。

本体系同时认识到：分类命名本身即是一种**文化翻译（cultural translation）**（Asad, 1986）。当“鬼”“神”“妖”等标签在古典文献与现代叙事之间迁移时，它们经历了**概念的跨语境旅行（traveling concepts）**（Said, 1983; Bal, 2002）。本模型旨在为这种“旅行”提供一张清晰的导航图。

## 📌 涉及神祇的说明

本体系只基于文艺作品中“神性角色”的叙事坐标描摹。本框架绝不涉及任何现存正统宗教的、拥有活态信徒的神祇。作者深知**真正的神圣性，远在人类的文化分析工具之外**。这只是一场思维实验，所有分析，仅限于对文学、影视、民间传说及数字叙事作品中所出现的、被角色或文本称为“神”的生灵，进行叙事功能上的**粗略坐标定位**。

## 🧬 核心发明：双轴模型（The Bi-Axial Model）

本体系放弃传统"鬼/妖/神/怪"的混沌命名与价值判断，改用**两个独立坐标轴**精准定位任何超自然实体。

如 Jeffrey Jerome Cohen 在 *Monster Theory: Reading Culture* (1996) 中所言，"怪物"本质上是文化的"他者"（the Other），是人认知边界的投射。本模型正是对这一"他者"进行系统化定位的工具。

### 第一轴：力量本源（The Source）

| 代码 | 名称 | 定义 | 典型表现 |
|:---|:---|:---|:---|
| **S-Nat** | 自然本源 | 由天地灵气、日月精华、自然现象所生 | 精怪、自然灵、自然拟人化神祇 |
| **S-Soul** | 灵魂本源 | 由执念、怨念、香火愿力所生 | 怨灵、香火神、地缚灵 |
| **S-Bio** | 生物本源 | 由DNA、细胞、病毒、血肉变异所生 | 丧尸、吸血鬼、感染体 |
| **S-Cos** | 异界本源 | 来自高维空间或外宇宙 | 外神、不可名状之物 |
| **S-Rule** | 规则本源 | 以“规则/条例/守则”为存在基础 | 规则怪谈污染源 |
| **S-Concept** | 概念本源 | 以“集体认知/叙事逻辑”为本体 | 怪谈意志、诅咒叙事体 |

**本源互斥性说明**：
- S-Nat：源于**自然物质世界**的能量转化
- S-Soul：源于**生命体精神活动**（执念/愿力）的凝结
- S-Bio：源于**生物学层面**的变异与感染
- S-Cos：源于**此世界之外**的维度或空间
- S-Rule：源于**规则系统本身**的本体性
- S-Concept：源于**集体认知**的具象化

### 第二轴：存在维度（The Dimension）

| 代码 | 名称 | 定义 | 典型表现 |
|:---|:---|:---|:---|
| **D-0** | 纯灵态 | 完全无形，不可肉眼直接观测 | 托梦、附身、感应的存在 |
| **D-1** | 半虚态 | 能显形但无物理实体 | 雾气、影子、海市蜃楼态 |
| **D-2** | 实体态 | 拥有完全物理肉体 | 可被物理攻击伤害 |
| **D-3** | 化身态 | 本体在高维，可投射分身 | 神明下凡、外神投影 |
| **D-Info** | 信息态 | 以信息/规则/概念形式存在 | 通过文字、视频、认知传播 |

**维度互斥性说明**：
- D-0：物理上**不可感知**
- D-1：物理上**可见但无实体**
- D-2：物理上**有形质**
- D-3：物理上**有分身/化身**
- D-Info：物理上**无形体，以信息媒介存在**

### 定位公式
> **每个超自然实体 = (Source坐标, Dimension坐标)**

*示例*：
- 古典怨灵 = **(S-Soul, D-1)**
- 生物型吸血鬼 = **(S-Bio, D-2)**
- 异界外神 = **(S-Cos, D-0/D-3)**
- 规则污染源 = **(S-Rule, D-Info)**

## 📌 关于“诡异”作为集合标签的说明

在当代中文网络文学中，“诡异”是一个**跨分类的集合标签**，覆盖 S-Soul（灵体型诡异）、S-Bio（实体型诡异）、S-Rule（规则型诡异）和 S-Concept（概念型诡异）中的多个节点。

“诡异”本身并非独立的分类，而是人类对“无法用日常逻辑解释的超自然现象”的统称。本模型将其视为**集合标签**而非**本源类别**。

## ⚠️ 本模型的边界

任何分类体系都有其适用边界。本模型明确以下限制：

1. **分类工具，非解释工具**：本模型能定位“什么坐标”，但不解释“为什么某个文化产生了这个实体”（那是历史学和民俗学的任务）。

2. **以叙事文本为主要依据**：本模型的坐标判定依赖于文本/影像中呈现的设定。口述传统、仪式实践中的超自然实体（如仪式中“附身”现象）可能无法被完整捕捉。

3. **不涵盖纯心理现象**：如幻觉、梦境产物、虚构妄想——这些属于心理学范畴，本模型不涉及。

4. **以人类文化为边界**：本模型适用于人类文化中产生的超自然概念，不涉及非人类文明的认知范畴。

5. **动态而非静态**：本模型承认分类本身是一种文化实践。随着新叙事形式的出现，模型的可扩展性（如新增本源或维度代码）是其设计的一部分。
