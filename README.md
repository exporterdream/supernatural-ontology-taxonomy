## 📌 关于"本体论"一词的说明

本体系所用的"本体论（Ontology）"取其**信息科学中的引申义**——即"对某一领域的概念及概念间关系的明确规范与分类"（Gruber, 1993），而非哲学形而上学意义上关于"存在本质"的研究。

此外，本体系认识到：概念的命名与分类本身就是一种**文化翻译（cultural translation）**行为（Asad, 1986）。当"鬼""神""妖"等标签从古典语境迁移到现代ACGN作品时，它们经历了**概念的跨语境旅行（traveling concepts）**（Said, 1983; Bal, 2002）。本模型的目标，正是为这种"旅行"提供一张清晰的导航图。

## 🧬 核心发明：双轴模型（The Bi-Axial Model）

本体系放弃传统"鬼/妖/神/怪"的混沌命名与价值判断，改用**两个独立坐标轴**精准定位任何超自然实体。

如 Jeffrey Jerome Cohen 在 *Monster Theory: Reading Culture* (1996) 中所言，"怪物"本质上是文化的"他者"（the Other），是人认知边界的投射。本模型正是对这一"他者"进行系统化定位的工具。

### 第一轴：力量本源（The Source）——决定"它由什么构成"

| 代码 | 名称 | 定义 | 典型案例 |
| :--- | :--- | :--- | :--- |
| **S-Nat** | 自然本源 | 由天地灵气、日月精华、自然现象所生 | 山神、树精、洪荒圣人 |
| **S-Soul** | 灵魂本源 | 由生物（尤其是人）的执念、怨念、香火愿力所生 | 厉鬼、城隍、土地公 |
| **S-Bio** | 生物本源 | 由DNA、细胞、病毒、血肉变异所生 | 丧尸、吸血鬼 |
| **S-Cos** | 异界本源 | 来自高维空间或外宇宙的不可知存在 | 克苏鲁、外神 |

### 第二轴：存在维度（The Dimension）——决定"它以什么形态显形"

| 代码 | 名称 | 定义 | 典型案例 |
| :--- | :--- | :--- | :--- |
| **D-0** | 纯灵态 | 完全无形，不可肉眼观测，仅能托梦或附身 | 克苏鲁本体、高阶天神 |
| **D-1** | 半虚态 | 能显形但无实体（如雾气、影子、海市蜃楼） | 聂小倩、贞子（录像带态） |
| **D-2** | 实体态 | 拥有完全物理肉体，可被物理攻击伤害 | 德古拉、丧尸、林正英僵尸 |
| **D-3** | 化身态 | 本体在高维，可投射一个或多个分身下界 | 神仙下凡、外神投影 |

### 最终定位公式
> **每个超自然实体 = (Source坐标, Dimension坐标)**

*示例*：
- 聂小倩 = **(S-Soul, D-1)**
- 德古拉 = **(S-Bio, D-2)**
- 克苏鲁 = **(S-Cos, D-0/D-3)**
- 孙悟空 = **(S-Nat, D-2 + D-3)**
